# Passkeys

Passkeys based authentication based on https://www.passkeys.com/guides

- **server** Node+express
- **client** Angular

## Getting started

```sh
pushd client
npm install
npm run build
popd
pushd server
npm install
npm start
```

You can use emulated authenticator in Chrome.
See: [WebAuthn: Emulate authenticators](https://developer.chrome.com/docs/devtools/webauthn/)
